Blockchain Teaching Tools

Interactive aids for an advanced blockchain analysis seminar. Pick one up, drive it live in front of the class, and let people change an input and watch the consequence.
Tool 01 · Bitcoin
UTXO Sandbox
Build a Bitcoin transaction by selecting unspent outputs from a wallet. Watch the selected UTXOs get consumed, a payment output and a change output get created, and the fee fall out as the difference. Includes scenario presets and a live transaction history.
Teaches: whole-output spending · change · fee = inputs − outputs
Open tool
Tool 02 · Tracing
Tracing-Method Calculator
Build a sequence of deposits and withdrawals on one commingled address, tag deposits as victim or clean, and run FIFO, LIFO, PIFO, pro-rata, and LIBR side by side on the same facts. The answers diverge — which is the point. Includes the worked examples from the methodology handout.
Teaches: method choice is a documented judgment · LIBR is a ceiling
Open tool
